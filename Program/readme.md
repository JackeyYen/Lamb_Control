# 使用说明：

    使用Arduino IDE 进行代码烧入测试：(使用PC进行)
         下载Arduino IDE
         接上Arduino UNO板
         打开.ino(该代码必须要在一个同名文件夹下，自行建立一个同名文件夹)
         选择好硬件版本与串口
         烧入
         可使用IDE提供的串口监控来查看运行状态
